# Project Tech Stack

## Frontend

* **HTML5** — base markup structure for web pages.
* **CSS3** — styling for UI components.
* **Bootstrap 5** — responsive CSS framework for building layouts quickly.

---

## Backend

* **Django** — main Python framework for server-side logic.
* **PostgreSQL** — relational database used for primary data storage.
* **psycopg** — official PostgreSQL adapter for Python, used by Django to connect to the database.

---

## Infrastructure

* **Docker** — containerization for isolating services (caddy, pgadmin, db, nginx).

---

## Code Quality

* **black** — tool for automatic Python code formatting.
* **pytest** — testing framework for writing and running tests.
